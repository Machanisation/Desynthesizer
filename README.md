# Desynthesizer

It is basic desynthesizer. It scans your inventory to find all items that can be desynthesized and then allows you to select which you want to desynthesize. 

Future plans: 

- Allow users to select how many items in a stack to desynthesize
- Add a button to quickly select all gear

# Note: For stacks, it will desynthesize everything 1 by 1. 

# How to use

Step 1: Select Desynthesizer in drop down and click on bot base Settings

Step 2: Click Scan Inventory in new window

Step 3: Select items you want to desynthesize

Step 4: Click start and watch it desynthesize it

Note: Sometimes it will fail to desynthesize an item, but after a few seconds, it will try again. So let it do its thing.
